# Telegram Ledger Chatbot

A simple Telegram chatbot that tracks expenses, categorises them automatically, and saves them into Supabase.

## Example Messages

```text
lunch 8.50
grab 12.40
shopee 51.70
zus coffee 5.90
```

## Commands

```text
/start
/summary
```

## Tech Stack

- Telegram Bot
- Supabase Database
- Supabase Edge Functions
- GitHub
